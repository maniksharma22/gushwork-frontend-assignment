# Gushwork Frontend Assignment

This project is a responsive product webpage built using **Vanilla HTML, CSS and JavaScript** following the provided Figma design.

## Features

- Pixel-aligned UI based on Figma
- Sticky header on scroll
- Interactive image carousel
- Image zoom preview on hover
- Fully responsive layout (desktop, tablet, mobile)

## Tech Stack

HTML5  
CSS3 (Flexbox & Grid)  
Vanilla JavaScript

## Project Structure

index.html – main webpage  
styles.css – styling  
script.js – interactions and carousel logic  
assets – images and icons
